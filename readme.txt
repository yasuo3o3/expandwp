広いディスプレイでGutenbergの左右ペイン枠を安全に拡張する最小実装。キーボードショートカット・ドラッグリサイズ対応。完全なアンインストール処理付き。

== Description ==

ExpandWPは、広いディスプレイ環境でWordPressのGutenbergエディタの左右ペインの幅を安全に拡張できる軽量プラグインです。

= 主な機能 =

* **左右ペインの拡張**: List Viewパネル（左）と設定パネル（右）を最大480pxまで拡張
* **キーボードショートカット**: Alt+,（左）、Alt+.（右）、Alt+0（リセット）
* **ドラッグリサイズ**: マウスでパネル幅をリアルタイム調整
* **自動モード**: パネルが開いたときに自動拡張
* **安全制約**: 編集キャンバスの最小幅1000pxを保護
* **RTL対応**: 右から左の言語環境にも対応

= 対象画面 =

* ブロックエディタ（投稿・固定ページ編集）
* サイトエディタ（フルサイト編集）
* カスタマイザー（追加CSS編集時）

= 動作モード =

* **手動**: ショートカットでのみ拡張（デフォルト）
* **自動**: 対象パネルが表示されたときに拡張
* **常時**: 画面読み込み時に即座に拡張

1440px未満のビューポートでは自動・常時モードは抑制され、手動操作のみ有効になります。

= アンインストール =

プラグインをアンインストールすると、以下のデータが完全に削除されます：

* 設定オプション（expandwp_options）
* パネル幅保存データ（expandwp_width_*）
* 関連する全ての一時オプション

データベースから完全に削除され、痕跡は残りません。

= キーボードショートカット =

* `Alt + ,`: 左パネル（List View）のトグル
* `Alt + .`: 右パネル（設定パネル）のトグル
* `Alt + 0`: 両パネルの幅をリセット

入力中（input/textarea/contenteditable）のときは自動的に無効化されます。

= 制約事項 =

* 編集キャンバスの最小幅1000pxを維持するため、必要に応じてパネル幅を自動調整
* 1440px未満のビューポートでは自動・常時モードは動作しません
* 内部UI要素の詳細なスタイルは変更せず、枠の拡張のみ実行
* 外部ライブラリを使用せず、バニラJavaScriptで実装

== Installation ==

1. プラグインファイルを `/wp-content/plugins/expandwp/` ディレクトリにアップロード
2. WordPress管理画面の「プラグイン」メニューからExpandWPを有効化
3. 「設定」→「ExpandWP」で動作モードを設定（オプション）

== Frequently Asked Questions ==

= 動作に必要なWordPressバージョンは? =

WordPress 5.9以降、PHP 7.4以降が必要です。Gutenbergエディタが有効である必要があります。

= キーボードショートカットが効かない =

入力フィールド（テキストエリア、コンテンツ編集可能な要素）にフォーカスがある場合は、既存機能との競合を避けるため無効化されます。

= パネル幅が勝手に縮小される =

編集キャンバスの最小幅1000pxを保護するため、必要に応じて自動調整されます。これは意図された動作です。

= 小さい画面でも使える? =

1440px未満のビューポートでは自動・常時モードは抑制されますが、手動でのトグル操作は可能です。

= プラグインを削除すると設定はどうなる? =

プラグインアンインストール時に設定データとパネル幅データは完全に削除されます。データベースから痕跡も残りません。
